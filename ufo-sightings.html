UFO Sightings Analysis

Team: Cidália Castro, Ana Carolina, Johab Santos

Date: March 2025

🛠️ Tools Used

MySQL · Power BI · DAX · Power Query · Excel

🎯 Main Objectives

  • Clean and normalize real-world semi-structured data
  • Design and populate a relational database using MySQL
  • Develop KPIs and advanced measures using DAX
  • Visualize spatial and temporal trends with interactive dashboards
  • Apply outlier detection and variable categorization

📊 Dashboard Highlights

  • Total and average duration of sightings
  • Top 5 countries and UFO shapes
  • Sightings by year, weekday, and duration category
  • Geographic map and Word Cloud from witness comments
